# 62、信号重载了，如何确定连接哪个信号？

采用函数指针确定连接哪个信号。 
```c++
// 假设QPushButton有如下重载信号（实际QPushButton没有这两个信号，仅为示例）
class MyButton : public QPushButton {
    Q_OBJECT
signals:
    void clicked();                // 无参版本
    void clicked(bool checked);    // 带bool参数的版本
};

// 在连接时
MyButton *btn = new MyButton;
QObject::connect(
    btn, 
    static_cast<void (MyButton::*)()>(&MyButton::clicked),  // 明确指定无参版本
    this, 
    &MyClass::onButtonClicked
);

// 连接带参数的版本
QObject::connect(
    btn, 
    static_cast<void (MyButton::*)(bool)>(&MyButton::clicked),  // 明确指定带参版本
    this, 
    &MyClass::onButtonChecked
);
```

qOverload
```c++
#include <QOverload>

// 连接无参信号
QObject::connect(
    btn, 
    qOverload<>(&MyButton::clicked),  // 无参版本
    this, 
    &MyClass::onButtonClicked
);

// 连接带参数的信号
QObject::connect(
    btn, 
    qOverload<bool>(&MyButton::clicked),  // 带bool参数版本
    this, 
    &MyClass::onButtonChecked
);
```

使用 Lambda 表达式作为中间层
```c++
// 连接无参信号
QObject::connect(btn, &MyButton::clicked, this, [](){
    // 处理无参信号的逻辑
});

// 连接带参数的信号（Lambda会根据参数匹配正确的重载）
QObject::connect(btn, &MyButton::clicked, this, [](bool checked){
    // 处理带参信号的逻辑
});
```
